Did JavaScript Function Run:

NOTE - this template is meant for simple HTML forms that are served via PHP (i.e. contact.php). It is a starting point for running server code (php most likely) based on whether the page was loaded in a browser. It can be used as is (as templates are used) if the form is submitted and processed on the same page i.e. action="<?php echo $_SERVER["PHP_SELF"]; ?>"; or altered to meet your needs. For help on getting started see the example below, or watch the YouTube Instructional Video.

How the Template Works

Overview:

The script file didJavaScriptFunctionRun.js is sourced in the HTML head, defining main variable - didJavaScriptFunctionRunCheck. It also makes an AJAX call to create the elements via PHP using scripts/make_random_file.php. The end result is a PHP variable - $didJavaScriptFunctionRun; that stores a value of 1 or 0. With that run code on the server accordingly.

Essentially the main goal of the template is to utulize a condition like the one below: 

 
    <?php
     if ( $didJavaScriptFunctionRunCheck == 1 ) { // DID load in browser
       // server's response
       echo "Form Submitted";
     } else { // did NOT load in browser
       echo " ";       
     }
    ?>
